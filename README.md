# QuantizedPyTorch (clean version)

## [Install](docs/install.md)
```bash
git clone https://github.com/wangying-ict/LLSQ.git

cd LLSQ
export PYTHONPATH=$PYTHONPATH:`pwd`
pip install -r requirements.txt
```
